# hscsm
R-package version 1.0.0:  display gene expression in transcriptome of MSCs from different tissues 








## package installation
```r
library(devtools)
install_github("cdesterke/hscsm")
```

## package usage
```r
library(hscsm)
data(csm)
csmbox(csm,"CXCL12")
```
